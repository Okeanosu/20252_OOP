-If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
Yes. As "8. Create Constructor method" said, it's called "Overloading method". So, as long as the parameter lists are different then it's okay.
for Example:
Create constructor method for DigitalVideoDisc:
- Create a DVD object by title
then:
- Create a DVD object by category, title and cost
we can see the Overloading method is working.

DVD String title and DVD String category will not work. If signature of the same name has different type then it will work.
